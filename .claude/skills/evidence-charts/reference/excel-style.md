# Excel 기본 차트 스타일 토큰

`scripts/excelchart.py`가 구현하는 룩의 명세. 헬퍼를 고치거나, 헬퍼가 못
그리는 특수 차트를 예외적으로 직접 그릴 때 이 토큰을 그대로 쓴다 — 목표는
"Office 기본 테마로 엑셀에서 방금 그린 차트"처럼 보이는 것.

## 색 (Office 기본 팔레트, 시리즈 순서 고정)

| 순서 | 색 | 용도 |
|---|---|---|
| 1 | `#4472C4` 파랑 | 첫 시리즈 (단일 시리즈면 이것 하나만) |
| 2 | `#ED7D31` 주황 | 둘째 시리즈 |
| 3 | `#A5A5A5` 회색 | 셋째 |
| 4 | `#FFC000` 금색 | 넷째 |
| 5 | `#5B9BD5` 연파랑 | 다섯째 |
| 6 | `#70AD47` 초록 | 여섯째 |

강조가 필요하면 색을 새로 만들지 말고 **강조 항목만 1번 파랑, 나머지를 3번
회색**으로 (엑셀 사용자들이 실제로 하는 방식). 기준선(한계선)은 `#C00000`
빨강 파선.

## 텍스트·프레임

- 폰트: Calibri (없으면 metric 호환 Carlito, 최후 DejaVu Sans)
- 제목: 13pt `#404040`, **가운데 정렬** (엑셀 기본), 꾸밈 없음
- 축 눈금·범례: 9pt `#595959`, 눈금선(tick mark) 없음
- 축 제목: 10pt `#595959`
- 그리드: 값 축 방향 **한쪽만** `#D9D9D9` 0.8pt (세로 차트 = 가로선만)
- 축선: `#BFBFBF` 0.8pt, 위·오른쪽 spine 제거
- 범례: 시리즈 2개 이상일 때만, **하단 중앙**, 테두리 없음
- 배경: 흰색, plot area 테두리 없음, 그림자·3D·그라데이션 절대 금지

## 기하

- 막대: gap width 219% (막대폭 ≈ 슬롯의 31%) — 엑셀 특유의 "가는 막대"
- 시리즈 겹침(overlap): −27%
- 선: 2.25pt, 마커는 데이터 포인트가 적을 때만(`markers=True`, 4pt 원)
- 데이터 레이블: 9pt `#595959`, 막대 끝 바깥쪽 (엑셀 'outside end')

## 각주

Source 각주는 그림 좌하단 바깥, 7.5pt 이탤릭 `#7F7F7F`,
`Source: <표기>` 형식 — 헬퍼가 자동으로 붙인다.

## 크기

PPT 3단 컬럼 셀 기준 폭 5–6.6", 높이 2.4–3.2", 200dpi. 슬라이드에서
축소되므로 9pt 미만 텍스트를 추가로 넣지 않는다.
