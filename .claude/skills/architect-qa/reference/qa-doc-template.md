## 별점 체계 공통 규칙

- **3단계 만점 3**: `★★★` / `★★☆` / `★☆☆`. 모든 별점은 각 QA의 **정량 bin**에
  대응해야 하며, bin 판정 근거 없는 별점은 무효.
- **평가 시점 표기**: 설계 예측 `(F)` / 실측 `(M)`.
- **근거 등급**: `A` 자체 실측 · `B` 문헌/벤치마크 인용 · `C` 구조 논증.
- **상한/도달 리스크 분리 표기**: `★★★ (상한) / ★☆ (도달 리스크)`.
